# week8-stabalize-app
Final Challenge - Stabilize the app for in class assignment which is a random quote assignment 
`main.js` — coordinates the app and handles events  
- `state.js` — stores and updates the current quote  
- `render.js` — updates the DOM to show the quote  
- `api.js` — fetches a random quote

# Test Plan
1. Check that clicking the button shows a new quote  
2. Check that `state.js` correctly updates the current quote  
3. Check that `render.js` updates the page with the current quote  
4. Test full app
